Contents

  • ai_compare_cell_type_labels_pairwise_adata_dict()

anndict.wrappers.ai_compare_cell_type_labels_pairwise_adata_dict#

anndict.wrappers.ai_compare_cell_type_labels_pairwise_adata_dict(adata, cols1, cols2, new_col_prefix='agreement', comparison_level='binary')[source]#

Compare cell type labels by finding unique combinations between labels in cols1 and cols2, applying the comparison, and mapping the results back to adata.obs.

Parameters:
adata AnnData

an AnnData.

cols1 list[str]

List of columns to compare against cols2.

cols2 list[str]

List of columns to compare with cols1.

new_col_prefix str (default: 'agreement')

The base name for the new comparison result columns.

comparison_level str (default: 'binary')

Determines which comparison function to use. Can be either 'binary' or 'categorical'. If 'binary', cell type labels are assessed for matching. If 'categorical', the quality of the label match is assessed as one of three levels: 'perfect', 'partial', or 'not-matching'.

Return type:

dict[tuple[str, str], DataFrame]

Returns:

Dictionary with keys as tuples of (col1, col2) and values as DataFrames with the comparison results.
