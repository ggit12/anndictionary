Annotating with AnnDictionary
===============================

AnnDictionary provides a suite of tools for annotating single-cell data and comparing the results. These notebooks demonstrate several of these tools. For the full list, see the documentation, including:

- `Annotate Cells <https://ggit12.github.io/anndictionary/api/annotate/cells/index.html>`_
- `Annotate Genes <https://ggit12.github.io/anndictionary/api/annotate/genes/index.html>`_
- `Plot <https://ggit12.github.io/anndictionary/api/plot/index.html>`_
